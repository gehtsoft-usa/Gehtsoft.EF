public property ValidationFailure.Path

Brief

The path to the value failed.

Declaration
C#
string  Path get; set;

Details

If the validator didn't call other validators via ValidateUsing() method, the path is equal to the name.

If other validaters are used, the property will consists of the full path of values, e.g. when property "X" is validated using other validator and the other validator created the failure with name "Y", the path will consists of "X.Y".
