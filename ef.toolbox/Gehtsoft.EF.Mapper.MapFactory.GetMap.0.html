static public method MapFactory.GetMap

Brief

Gets the map.

Declaration
C#
Map<TFrom,TTo>  GetMap<TFrom,TTo> (bool createIfNotExists = true)

Details

If map does not exists, the method automatically creates it unless you request not to passing false as a parameter. If map creation is requested, the map automatic initialization must be possible (i.e. it must be a map between an entity and a model or a class and a model).

See MapFactory.CreateMap() for more details about the map creation.
