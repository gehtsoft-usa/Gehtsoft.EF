static public method MapFactory.CreateMap

Brief

Creates a new map.

Declaration
C#
Map<TFrom,TTo>  CreateMap<TFrom,TTo> ()

Details

If it is possible to initialize map automatically (e.g. this is an attribute-based entity to model, model to entity, class to model or model to class mapping), it will be initialized.

Please note that properties attributed using DoNotAutoMapAttribute won't be mapped.

Otherwise an empty map will be returned. You can refine the map definition using MapPropertiesByName extension method of the map or using map's methods.
