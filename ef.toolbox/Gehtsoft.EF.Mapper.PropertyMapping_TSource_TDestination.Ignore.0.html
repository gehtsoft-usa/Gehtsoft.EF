public method PropertyMapping of (TSource,TDestination).Ignore

Brief

Set the rule predicate that makes it never run.

Declaration
C#
PropertyMapping<TSource,TDestination>  Ignore ()

Details

Please note that the rule has only ONE predicate, so any subsequence call of the methods that sets the predicate will remove the previously set predicate.

There are also an extension method for IEnumerable of PropertyMapping which applies Ignore for all elements of the enumeration.
