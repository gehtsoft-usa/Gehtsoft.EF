protected method ExpressionCompiler.WalkExpression

Brief

Scans the expression and returns its JavaScript presentation.

Declaration
C#
string  WalkExpression (Expression expression)

Parameters
expression

Details

Wherever it is possible, the function attempts to calculate the expression value and put the resulting constants inside of the whole expression tree.

Otherwise, it calls one of the other methods (see protected methods started with Add) for particular type of the expression.
