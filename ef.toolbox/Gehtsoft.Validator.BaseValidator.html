class BaseValidator

Parents

IBaseValidator

Brief

Basic validator.

Details

This class is designed rather for automated constructor of validators. To define validation rules in the source code the AbstractValidator<T> is highly recommended.

The base validator automatically creates validation rules for all validation attributes applied to the properties of the type passed to the constructor.

The type defined in the Gehtsoft.Validator.dll assembly. The namespace is Gehtsoft.Validator.
