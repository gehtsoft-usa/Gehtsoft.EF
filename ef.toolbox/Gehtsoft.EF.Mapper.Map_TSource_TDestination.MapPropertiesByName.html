public method Map of (TSource,TDestination).MapPropertiesByName

Brief

Adds mapping rules associating source and destination type by property names.

Declaration
C#
void  MapPropertiesByName (bool onlyValueTypes = false, string[] propertyIgnoreList = null, Type[] typeIgnoreList = null)

Parameters
onlyValueTypes

If this flag is set, only value properties (i.e. numbers, string, dates, booleans) will be mapped.
propertyIgnoreList

The list of the source type or destination type property names to be excluded from the mapping.
typeIgnoreList

The list of the types to be excluded from the mapping.

Details

Please note that properties attributed using DoNotAutoMapAttribute won't be mapped.
