class SqlFunction

Parents

object

Brief

The list of SQL functions to be used in the query-related expression.

Details

Using this function forces the compiler to execute the function on the server side. For example, is string.Upper() is used and the arguments can be calculated locally, it will be calculated locally. But if SqlFunction.Upper() is used the function will always be executed on the server side.
