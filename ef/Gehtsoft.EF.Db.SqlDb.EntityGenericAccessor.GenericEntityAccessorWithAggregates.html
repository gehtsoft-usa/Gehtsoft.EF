Brief

The CRUD accessor for an entity with aggregated.

Parameters
T
TKey

Details

Aggregate is an entity related to the main entity with one to many. An example of aggregates would be order details for an order.

The accessor modifies some operations, e.g. when an aggregating entity is deleted, all aggregates are deleted automatically.
