class EntityActionDelegate

Parents

MulticastDelegate

Brief

The delegate type of the action called when entity is created.

Details

The delegate prototype is:

public delegate void EntityActionDelegate(SqlDbConnection connection);

The action is invoked only when CreateEntityController is used to create the entity.

To set the action use OnEntityCreateAttribute , OnEntityDropAttribute , OnEntityPropertyCreateAttribute or OnEntityPropertyDropAttribute .
