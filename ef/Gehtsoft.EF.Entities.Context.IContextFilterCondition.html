interface IContextFilterCondition

Brief

A single condition inside the filter expression.

Details

Example:

condition.Property("a").Is(CmpOp.Eq).Value(1);
condition.Property("a").Is(CmpOp.Le).Property(b);

See also IContextFilter .

Use EntityFilterConditionExtension for more readable definition of the conditions.

Public Methods

Is

Sets the operation.

Property

Adds a property to the condition.

Value

Adds the value of the condition.
