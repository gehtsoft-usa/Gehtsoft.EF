class SelectEntitiesQuery

Parents

SelectEntitiesQueryBase

Brief

The query to select the entity and all dependent entities (dictionaries).

Details

Use EntityConnectionExtension.GetSelectEntitiesQuery to get an instance of this query.

In cause you don't need complete tree/all columns, use SelectEntitiesQueryBase to minimize DB load and traffic between the application and database.

The object instance must be disposed after use. Some databases requires the query to be disposed before the next query may be executed.
