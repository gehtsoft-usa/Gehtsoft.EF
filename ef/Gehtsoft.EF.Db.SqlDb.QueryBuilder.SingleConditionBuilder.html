class SingleConditionBuilder

Parents

object

Brief

Builder for a single condition.

Details

The builder for a single condition is returned by ConditionBuilder .

Define a condition as a following:

  • Set the left side

  • Set the operation

  • Set the right side

query.Where.Property(table["column"]).Is(CmpOp.Eq).Parameter("c1");

Use this class to define one comparison or check within a complex condition. See SingleConditionBuilderExtension to easier and more readable definition of the condition.
