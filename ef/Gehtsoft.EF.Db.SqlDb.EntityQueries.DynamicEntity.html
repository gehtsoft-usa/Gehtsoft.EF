class DynamicEntity

Parents

System.Dynamic.DynamicObject

Brief

The base class for dynamic entities.

Details

Note: dynamic entities will not be discovered by default by EntityFinder . In order to enable discovery them in finder, call AllEntities.EnableDynamicEntityDiscoveryInEntityFinder method.

To implement a dynamic property, override DynamicEntity.EntityAttribute property and DynamicEntity.InitializeProperties method.

Public Properties

EntityAttribute

Override the method to return the entity attribute.

ObsoleteEntityAttribute

Override the method to return the obsolete entity attribute

Properties

Returns the list of the properties metadata.
