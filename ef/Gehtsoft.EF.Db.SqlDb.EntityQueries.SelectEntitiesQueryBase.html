class SelectEntitiesQueryBase

Parents

ConditionEntityQueryBase

Brief

The base query to all entity select operation.

Details

Use EntityConnectionExtension.GetSelectEntitiesQueryBase to get an instance of this object.

You can use this query type directly to fine tune the resulset or use SelectEntitiesCountQuery or SelectEntitiesQuery .

The object instance must be disposed after use. Some databases requires the query to be disposed before the next query may be executed.
