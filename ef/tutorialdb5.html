Data Modification Query Builders

INSERT

The InsertQueryBuilder creates the SQL engine specific query for the INSERT and defines all parameters. The parameters has example the same name as the columns to be inserted. Please note that the query builder takes care about proper using of suffixes for the parameters (e.g. @ for MSSQL or : for Oracle) so wherever you are working with the SQL builder you can simply use the parameter name without any prefixes. It also handles autoincrement columns and defines output parameter that accepts auto-assigned value.

using (SqlDbQuery query = connection.GetQuery(connection.GetInsertQueryBuilder(my_table)))
{
    query.BindParam("name", "name1");
    query.BindParam("volume", 112.5);
    query.ExecuteNoData();
    int newid = query.GetParamValue<int>("id");
}

DELETE

The DeleteQueryBuilder creates the SQL engine specific query to delete rows from the table. In this example we look at the simplest version that just deletes one row by its identifier. Use DeleteById() method to create such query. The parameter with the same name as a primary key column will be created.

Example: Deleting a row Show

Example: Deleting a row Hide

DeleteQueryBuilder deleteBuilder = connection.GetDeleteQueryBuilder(my_table);
deleteBuilder.DeleteById();
using (SqlDbQuery query = connection.GetQuery(deleteBuilder))
{
    query.BindParam("id", 1);
    query.ExecuteNoData();
       
}

UPDATE

The UpdateQueryBuilder create the SQL engine specific query to update rows in the table. In this example we look at the simplest version that just updates one row by its identifier. Use UpdateById method to create such query. Use AddUpdateColumn to add one column to update. Use AddUpdateAllColumns to add all columns except primary key to update. Both methods create parameter(s) with the same name as the column name to be updated.

Example: Updating a row Show

Example: Updating a row Hide

UpdateQueryBuilder updateBuilder = connection.GetUpdateQueryBuilder(my_table);
updateBuilder.UpdateById();
updateBuilder.AddUpdateColumn(my_table["name"]);
using (SqlDbQuery query = connection.GetQuery(updateBuilder))
{
    query.BindParam("id", 1);
    query.BindParam("name", "new name for row 1");
    query.ExecuteNoData();
       
}
