class AlterTableQueryBuilder

Parents

object

Brief

The query builder for the ALTER TABLE command.

Details

Use SqlDbConnection.GetAlterTableQueryBuilder to create an instance of this object.

Please note that this builder is the only builder which is not derived from AQueryBuilder . Because of the specified nature of ALTER TABLE command, it returns a sequence of the commands instead of single query.

Public Methods

GetQueries

Get queries to perform requested operations.
