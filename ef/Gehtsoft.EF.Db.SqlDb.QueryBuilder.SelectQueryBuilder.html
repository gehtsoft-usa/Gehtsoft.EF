class SelectQueryBuilder

Parents

QueryWithWhereBuilder

Brief

The query builder for SELECT command.

Details

Use SqlDbConnection.GetSelectQueryBuilder to create an instance of this object.

You can also use to bind the resultset of a select query to a runtime object.

Public Properties

Distinct

The condition for HAVING clause.

The maximum number of the first rows to return.

QueryTableDescriptor

Returns the table descriptor that represents the resultset of the query.

Resultset

The resultset.

Skip

The number of rows to skip from the beginning of matching recordset.
