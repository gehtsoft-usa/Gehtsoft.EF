public method UnionQueryBuilder.AddQuery

Brief

Adds a query to the union.

Declaration
C#
void  AddQuery (SelectQueryBuilder builder, bool distinct)

Parameters
builder

The query to add
distinct

A flag indicating whether the union should take only distinct record (true) or all record, including duplicates (`false)

Details

The query must:

  • Have the same number of columns

  • The columns should be name the same way

  • The columns should have the same data type

  • Order should not be defined.
