Queries

There are a few entity queries that can be created using EntityConnectionExtension extension class.

The methods of this class creates one of subclasses of EntityQuery classes. An entity query is a disposable object and must be disposed after use and before the connection to which the query belongs is disposed.

An entity queries can be executed using method Execute.

Depending on the query it may have a parameter if the query operates on a single instance of the entity (e.g. insert, update or delete).
