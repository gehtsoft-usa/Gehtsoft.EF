class EntityQueryConditionBuilder

Parents

object

Brief

The condition builder for WHERE and HAVING clauses of entity queries.

Details

Each individual condition in the expression is build using SingleEntityQueryConditionBuilder .

Use EntityQueryConditionBuilderExtension to create an easier to read conditions.
