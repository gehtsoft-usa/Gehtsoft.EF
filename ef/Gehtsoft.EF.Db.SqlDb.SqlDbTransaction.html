class SqlDbTransaction

Parents

object

IDisposable

Brief

The database transaction.

Details

Use SqlDbConnection.BeginTransaction to create a transaction.

The transaction is considered ended when the object is disposed. The transaction should be committed explicitly, otherwise the driver will roll it back at the disposal.

Public Methods

Commit

Commits the transaction.

CommitAsync

Commits the transaction asynchronously.

Dispose

Disposes the transaction object.

Rollback

Rolls the transaction back.

RollbackAsync

Rolls the transaction back asynchronously.
