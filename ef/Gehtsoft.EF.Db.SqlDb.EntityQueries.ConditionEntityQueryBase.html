class ConditionEntityQueryBase

Parents

EntityQuery

Brief

The base class for all queries with a condition.

Details

This is a base abstract base class. Use MultiUpdateEntityQuery , MultiDeleteEntityQuery , SelectEntitiesQueryBase , SelectEntitiesCountQuery or SelectEntitiesQuery instead.

The object instance must be disposed after use. Some databases requires the query to be disposed before the next query may be executed.
